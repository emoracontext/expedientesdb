7.2 Solicitud de Justificantes de Gestoría


Permite solicitar el justificante de gestoría para obtener documento que sustituye el permiso de circulación mientras se realiza un trámite.


7.2.1 Request


URI Método Descripción

/justificantes

POST

Envía una nueva solicitud si no existe o actualiza la existente si ya existe una solicitud idéntica


Idempotente


Devuelve un número único asignado a la Solicitud

/justificantes/{solicitud_id}

GET

Devuelve el estado de la solicitud

/justificantes/{solicitud_id}?documentos=true

GET

Devuelve el justificante de gestoría oficial generado como resultado del trámite solicitado.
