6 REST

El API de Expedientes dealerBest es un API REST que emplea los métodos HTTP existentes para crear (POST), consultar (GET) y eliminar (DELETE) registros de Solicitudes.


Los métodos pueden actuar sobre registros individuales o colecciones de registros. La tabla siguiente detalla la actuación de los métodos en cada caso de uso.

El esquema básico de la sintaxis para las funciones de este API es:

https://host/namespace_de_recursos[/id][/verbo][?parametro1=valor1][... ?parametroX=valorX]


Los parámetros en el URL solo se admitirán en los métodos GET o DELETE y únicamente como se describe en este manual de referencia. Los parámetros de los métodos POST y PUT serán pasados en cuerpo del Request en formato JSON.


El API estará publicado y se podrá consultar su última versión en http://expedientes.dealerbest.com/webapi.
