9 Formatos de Datos

Input

Al crear solicitudes será imprescindible enviar los datos en formato JSON con el tipo de contenido HTTP:

Context-type: application/json


Output

Los datos devueltos por la aplicación se enviarán en una estructura JSON con el tipo de contenido HTTP:

Context-type: application/json


Los archivos anexos se podrán enviar en formato PDF (application/pdf), JPEG (image/jpeg) o PNG (image/png).


No se admiten datos en formato XML
