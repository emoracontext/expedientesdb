12 Anexos – Archivos adjunto

Muchos tramites requieren de documentación de apoyo escaneada. Estos documentos deben ser enviados como archivo binario en uno de los formatos soportados (JPG, PNG o PDF).


El número de archivos de apoyo que se anexan a una solicitud dependerá del tipo de trámite. Se podrá requerir ficheros como los documentos de identidad del titular o comprador, identificación fiscal de la empresa, permiso de circulación, ficha técnica, etc…


Los archivos se podrán anexar a una Solicitud existente mediante su número de Solicitud.


Los archivos se anexarán enviando un HTTP Request de tipo POST, codificados en formato base64- encoded, en un método POST.


El proceso a seguir para Anexar archivos a un expediente es:

1. Enviar la Solicitud de trámite mediante un HTTP POST Request y obtener un Número de Solcitud enviado por el servidor en el HTTP Response

2. Enviar un HTTP POST Request con el Número de Solicitud, el nombre de archivo, tipo de documento y el archivo binario con codificación base64.

Para sustituir una archivo adjunto se ha de volver a enviar el archivo un HTTP POST Request con el mismo Número de Solicitud y mismo nombre de archivo.


Si el archivo es anexado correctamente se devolverá un HTTP Response 200 (OK)


Para eliminar un archivo adjunto de un expediente se ha de enviar un http DELETE Request con el Número de Solicitud y el nombre de archivo que se quiere eliminar.


Si el archivo es eliminado correctamente se devolverá un HTTP Response 200 (OK)
