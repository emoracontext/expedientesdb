5 Definiciones


Solicitud: la plataforma de Expedientes de Dealerbest ha sido desarrollada como punto de intercambio de datos para tramitar expedientes de vehículos con la Dirección General de Tráfico. Los Colaboradores de dealerBest interactuarán con la plataforma envidando Solicitudes. Pueden enviar Solicitudes de Trámites o Solicitudes de Consulta.


No obstante, en el contexto de este documento la palabra “solicitud” puede tener dos significados. El vocablo “solicitud” es una palabra clave en la terminología de una aplicación web al tratarse de la traducción a nuestro idioma del “request” definido en el protocolo HTTP RFC-2616.


Para distinguir la solicitud HTTP de una Solicitud de trámite adoptaremos el siguiente convenio a lo largo del presente documento:

• Se escribirá Solicitud con la primera letra mayúscula cuando nos refereimos a las solicitudes de trámites en la plataforma.

• Cuando nos referimos a las solicitudes del protocolo http emplearemos el término anglosajón, “Request" o “HTTP request” tal como se define en el RFC-2616


Siguiendo esta convención, a lo largo de este documento, utilizaremos la nomenclatura descrita en RFC- 2616 para referiremos a la respuesta enviada por el servicio de Expedientes.
