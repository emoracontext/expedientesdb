12.2 Códigos de Respuesta http


Los códigos de estatus HTTP se dividen en las siguientes categorías: 1XX: Informativo – referentes a información a nivel de protocolo.

2XX: Éxito – la petición se cursó con éxito

3XX: Redirección – el cliente debe tomar alguna acción adicional para completar la petición 4XX: Error de Cliente – Petición fallida debido a un error del cliente

404 – Petición no válida 401 – No autorizado 403 – Acceso denegado

404 – No encontrado – el recurso solicitado no existe 405 – Método no permitido


5XX: Error del Servidor – La petición ha fallado por un error que se ha producido en el servidor.
